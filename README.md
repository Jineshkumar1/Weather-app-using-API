# J's Weather App
## Simple Weather App using HTML , CSS , and JavaScript  

Used Weather API by [API-Ninjas FREE Weather](https://rapidapi.com/apininjas/api/weather-by-api-ninjas/) 

Data Fetched using the API : 

Cloud_pct  
Temperature  
Feels Like  
Humidity  
Min_temp  
Max_temp  
Wind_speed   
Wind_degrees
Sunrise  
Sunset
